Gestión de Historiales

Pantalla inicial que permite el acceso a la aplicación. Solo un usuario registrado podrá hacer uso de sistema. Para acceder se debera introducir el usuario y la contraseña en los campos correspondientes. Una vez introducidos los datos se pulsa el boton aceptar para comprobar que es correcto y acceder al sistema. Si se desea salir de la aplicación pulsar el boton cancelar o cerrar la ventana directamente.

Usuario

Campo correspondiente al correo electronico asignado en la aplicacion como usuario del sistema.

Contraseña

Campo correspondiente con la contraseña asociada al correo electronico del ususario que intenta acceder a la aplicacion.
